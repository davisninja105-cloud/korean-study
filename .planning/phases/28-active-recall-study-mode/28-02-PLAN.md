**lib/active-prompt.ts contract (implement exactly this shape):**

```typescript
export type ActiveFace =
  | { face: 'passive-degrade' }                      // real card, state <= 1 (D-10)
  | { face: 'word-production'; prompt: string }      // practice card OR matured zero-sentence (D-04)
  | { face: 'sentence-production'; prompt: string }  // matured card with a chosen sentence

export function deriveActiveFace(
  card: { front: string; back: string },
  chosenSentence: { translation: string } | null,
  isNewCard: boolean,
  isPractice: boolean,
): ActiveFace
```

Precedence (order matters — a practice card is also "new" by the isNewCard computation since it has no review):
1. `isPractice` → word-production, prompt = card.back (BEFORE the new-card check)
2. `isNewCard` → passive-degrade (BEFORE the null-sentence check — a new zero-sentence real card must degrade, not get a production prompt for a never-seen word; Core Value)
3. `chosenSentence === null` → word-production, prompt = card.back
4. otherwise → sentence-production, prompt = chosenSentence.translation

Module conventions (mirror lib/sentence-selection.ts): JSDoc header stating the precedence + "Used by:" list + purity guarantee line; minimal structural param types (NOT CardDTO); named exports only; no 'use client'; no Prisma/Node imports.
